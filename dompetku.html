Panduan Deploy ke Internet
Ikuti langkah berikut agar web ini bisa diakses dari HP & laptop manapun, gratis.
Step 1 — Buat akun GitHub (gratis)
1
Daftar di GitHub

Buka github.com → klik "Sign up" → isi email & password → verifikasi email.

2
Buat repository baru

Klik tombol "+" di pojok kanan atas → "New repository" → beri nama dompetku → klik "Create repository".

3
Upload file index.html

Di halaman repository, klik "uploading an existing file" → drag & drop file index.html ini → klik "Commit changes".

Step 2 — Deploy ke Vercel (gratis selamanya)
4
Daftar di Vercel

Buka vercel.com → klik "Sign Up" → pilih "Continue with GitHub" → izinkan akses.

5
Import project

Di dashboard Vercel, klik "Add New → Project" → pilih repository dompetku → klik "Deploy". Tunggu 1-2 menit.

6
Web kamu sudah online!

Vercel akan memberikan link seperti dompetku.vercel.app — buka dari HP manapun, kapanpun. Gratis!

Step 3 — Tambah database Supabase (opsional, data tidak hilang)
Saat ini data tersimpan di browser (localStorage). Jika mau data tersimpan permanen & bisa diakses dari HP berbeda, tambahkan Supabase.
7
Daftar Supabase

Buka supabase.com → "Start your project" → daftar dengan GitHub → buat project baru → pilih region "Southeast Asia".

8
Salin API Key

Di Supabase: Settings → API → salin Project URL dan anon public key. Kirimkan ke aku, nanti aku buatkan versi dengan database penuh.

📱 Tips: Setelah deploy, buka web di HP → tekan menu browser → "Add to Home Screen" → web ini akan terasa seperti aplikasi HP!
